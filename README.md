# 🌏 Visualization-of-Population-Problems
2023-2학기 세종대학교 데이터시각화 수업 보고서

## 💡 Background
* 대한민국의 인구문제(고령화, 저출산)
* 혼인률 감소
* 젠더 갈등 증가
## 🛠️ Objective
* 본 보고서는 대한민국의 **인구(출산) 문제**를 다루며, 이러한 문제의 주요 원인 중 하나로 감소한 대한민국의 **혼인율**을 탐구합니다.
* **혼인율의 감소**와 관련된 주요 원인 중 하나로 남녀 간의 갈등, 일명 **"젠더 갈등"** 을 주제로 다루고자 합니다. 
* 현대 사회에서의 남녀 간 갈등의 양상을 파악하기 위해 대학생 커뮤니티 **"에브리타임"** 의 게시글을 크롤링하여 혐오표현의 사용 빈도를 조사합니다. 이를 통해, 우리 주변에서의 **남녀 갈등의 현황**을 보다 구체적으로 **분석**해보려 합니다.
